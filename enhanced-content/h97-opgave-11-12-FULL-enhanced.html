Uitwerking volgens 5-stappen methodologie

Opgave 11: 3 dB punt RC-filter

Stap 1: Analyseer de gegeven informatie

RC hoogdoorlaatfilter:

  • Serie RC: condensator blokkeert DC, doorlaat AC
  • 3 dB punt: -3 dB demping = factor 1/√2 ≈ 0,707
  • Bij 3 dB punt: faseverschuiving = 45°
  • R = 50 Ω
  • Gegeven: "De stroom moet voorlopen op de spanning, dus er moet een C zitten"

Holmes referentie: H 9.7 - RC-filters en 3dB punten

Stap 2: Bepaal toe te passen theorie

3 dB punt conditie:

Bij het 3 dB punt (hoekfrequentie) geldt voor een RC-filter:

XC = R (bij 45° faseverschuiving)

Uitleg:

Bij 45° faseverschuiving in een RC-serie is Z = R - jR (gelijke resistieve en capacitieve delen), wat resulteert in:

  • |Z| = R√2
  • Verzwakking = R/(R√2) = 1/√2 ≈ 0,707 = -3 dB
  • φ = arctan(-R/R) = arctan(-1) = -45°

Capacitantie uit XC:

C = 1 2πfR
Stap 3: Algebraïsche afleiding

Bij 3 dB punt geldt:

ZC = − 50j Ω

Dus:

1 2πfC = 50

Oplossen naar C:

C = 1 2πf · 50 = 1 100πf

Opmerking: Frequentie f is niet expliciet gegeven in de vraag, maar de formule toont de relatie. Bron geeft: C = 0,318 μF.

Als we f afleien uit bron-antwoord:

C = 0,318 μF, R = 50 Ω:

f = 1 2πRC = 1 2π · 50 · 0,318 × 10−6 ≈ 10 kHz
Stap 4: Numerieke conclusie

Antwoord volgens bron:

C = 0,318 μF

Bij deze C-waarde en R = 50 Ω:

  • 3 dB punt (f-3dB) ≈ 10 kHz
  • Tijdconstante τ = RC = 50 × 0,318×10-6 = 15,9 μs
  • Hoekfrequentie ωc = 1/τ = 1/(RC) ≈ 62,9 krad/s
Stap 5: Verificatie en fysische interpretatie

Fysische betekenis 3 dB punt:

Het 3 dB punt is de hoekfrequentie waar:

  • Uitgangsvermogen = ½ × ingangsvermogen (-3 dB)
  • Uitgangsspanning = 1/√2 × ingangsspanning (≈ 0,707)
  • Faseverschuiving = 45°
  • XC = R (gelijke resistieve en reactieve delen)

RC hoogdoorlaatfilter:

  • f < f-3dB: Sterk gedempt (C blokkeert)
  • f = f-3dB: -3 dB demping (overgang)
  • f > f-3dB: Minimaal gedempt (C doorlaat)

Opgave 12: R en L van spoel bepalen

Stap 1: Analyseer de gegeven informatie

Meetopstelling:

  • Spoel (L + Rspoel in serie, intern)
  • Externe weerstand: Rext = 10 Ω in serie
  • Stroom: I = 0,15 A = 150 mA
  • Twee condities:

Conditie 1:

  • Uspoel = 9,7 V (spanning over L + Rspoel)

Conditie 2:

  • Utotaal = 10 V (over alles: Rext + L + Rspoel)
  • Dit impliceert: Uext = I × Rext = 0,15 × 10 = 1,5 V

Holmes referentie: H 9.7 - Spoelkarakterisering

Stap 2: Bepaal toe te passen theorie

Impedantie-driehoeken opstellen:

Voor spoel alleen:

Uspoel2 = UR2 + UL2

Voor totale schakeling:

Utotaal2 = ( UR,ext + UR,spoel ) 2 + UL2
Stap 3: Algebraïsche afleiding

Gegeven spanningen:

  • Uspoel = 9,7 V
  • Utotaal = 10 V
  • UR,ext = 1,5 V

Opstellen vergelijkingen:

Vergelijking 1 (alleen spoel):

UR2 + UL2 = 9,72 = 94,09

Vergelijking 2 (totaal):

( UR + 1,5 ) 2 + UL2 = 102 = 100

Elimineer UL²:

Van vergelijking 2: UL² = 100 - (UR + 1,5)²

Van vergelijking 1: UL² = 9,7² - UR²

Gelijkstellen:

100 − ( UR + 1,5 ) 2 = 9,72 − UR2

Uitwerken:

100 − UR2 − 3UR − 2,25 = 94,09 − UR2

Vereenvoudigen (UR² valt weg):

100 − 3UR − 2,25 = 94,09
3UR = 100 − 2,25 − 94,09 = 3,66

Oplossing UR:

UR = 3,66 3 = 1,22 V

Bereken Rspoel:

Rspoel = UR I = 1,22 0,15 = 8,13 Ω

Bereken UL:

UL = 9,72 − 1,222 = 94,09 − 1,49 = 92,60 = 9,623 V

Bereken XL en L:

ZL = UL I = 9,623 0,15 = 64,15 Ω

Bij f = 100 kHz (blijkt uit bron):

L = ZL 2πf = 64,15 2π · 100000 = 102 μH
Stap 4: Numerieke verificatie

Controle vergelijking 1:

1,222 + 9,6232 = 94,10 = 9,70 V ✓

Controle vergelijking 2:

( 1,22 + 1,5 ) 2 + 9,6232 = 7,40 + 92,60 = 100 = 10 V ✓
Stap 5: Verificatie en fysische interpretatie

Fysische betekenis:

Spoelkarakterisering methode:

Door metingen met en zonder externe R kunnen we de interne parameters van een spoel bepalen:

  • Rspoel = wikkelweerstand (koperverliezen)
  • L = inductantie (energieopslag in magneetveld)

Kwaliteitsfactor spoel:

Q = XL Rspoel = 64,15 8,13 = 7,89

Q ≈ 8 is een redelijke kwaliteitsfactor voor een kleine spoel bij 100 kHz.

Praktische toepassing:

Deze meetmethode wordt gebruikt voor:

  • Spoelkarakterisering in productie
  • Kwaliteitscontrole van RF-spoelen
  • Bepalen van verliezen in transformatoren
  • Filter-ontwerp (Q bepaalt bandbreedte)

Antwoorden:

Opgave 11:

C = 0,318 μF

Bij R = 50 Ω geeft dit 3 dB punt bij ~10 kHz (XC = R = 50 Ω)

Opgave 12:

Rspoel = 8,13 Ω

L = 102 μH (bij f = 100 kHz)

Kwaliteitsfactor: Q = XL/R ≈ 7,89

Fysisch: Bij het 3 dB punt van een RC-filter zijn de resistieve en reactieve delen gelijk (XC = R), wat resulteert in 45° faseverschuiving en 1/√2 verzwakking. De spoelkarakterisering gebruikt twee meetpunten om zowel de wikkelweerstand (8,13 Ω) als de inductantie (102 μH) te bepalen, wat een kwaliteitsfactor Q ≈ 8 geeft - typisch voor kleine RF-spoelen.
