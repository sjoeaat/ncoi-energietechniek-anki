In dit netwerk is de stroom door de spanningsbron U1 gelijk aan:

a: I1=−I2−Ic

b: I1=−(I2+Ic+Id)

c: I1=+(I2+Ic+Id)

d: I1=−(Ib+Ic+Id)

Uitwerking
Stap 1: Analyseer de schakeling en gegeven informatie

Deze vraag test de Kirchhoff knooppuntwet (KCL) toegepast op een knooppunt met een spanningsbron.

Element Beschrijving
Spanningsbron U₁ Ideale spanningsbron met stroom I₁
Knooppunt Waar spanningsbron en weerstanden samenkomen
Uitgaande stromen I₂, Ic, Id door de weerstanden
Gezochte stroom I₁ door de spanningsbron

Theoretische basis: Kirchhoff knooppuntwet (KCL): de som van alle stromen naar/van een knooppunt is nul (Holmes §1.8).

Belangrijk: De stroom door een spanningsbron wordt niet door de spanningsbron zelf bepaald, maar door de rest van het netwerk via KCL.

Stap 2: Identificeer alle stromen bij het knooppunt

Bij het knooppunt waar U₁ aansluit zijn de volgende stromen aanwezig:

  • I1: stroom door de spanningsbron (richting nog te bepalen)
  • I2: stroom door R₂ (weg van knooppunt)
  • Ic: stroom door Rc (weg van knooppunt)
  • Id: stroom door Rd (weg van knooppunt)

Tekenconventie:

  • Stromen naar het knooppunt: positief
  • Stromen weg van het knooppunt: negatief

Als we aannemen dat I₂, Ic en Id weggaan van het knooppunt, dan moet I₁ naar het knooppunt stromen om de balans te behouden.

Stap 3: Pas Kirchhoff knooppuntwet toe

De som van alle stromen naar een knooppunt is nul. Stel dat I₁ naar het knooppunt stroomt (positief) en I₂, Ic, Id weg van het knooppunt (negatief):

I1 − I2 − Ic − Id = 0

Of anders geschreven met de conventie dat uitgaande stromen positief zijn:

I1 + I2 + Ic + Id = 0

waarbij I₁ de ingaande stroom is (dus negatief als stroom uit de bron komt).

Stap 4: Los op naar I₁

Isoleer I₁ in de vergelijking:

I1 = − I2 − Ic − Id

Of compacter geschreven:

I1 = − ( I2 + Ic + Id )

Conclusie: Het juiste antwoord is b.

Stap 5: Verificatie en fysische interpretatie

Dimensionale controle:

[I1] = [A] + [A] + [A] = [A]

Dit klopt ✓

Waarom andere antwoorden fout zijn:

Antwoord a: I1=−I2−Ic

  • Fout: Negeert de stroom Id
  • Dit zou alleen kloppen als Id = 0, wat niet algemeen geldt

Antwoord c: I1=+(I2+Ic+Id)

  • Fout: Positief teken suggereert dat I₁ in dezelfde richting stroomt als I₂, Ic, Id
  • Dit zou betekenen dat stroom uit het "niets" verschijnt - geen conservatie van lading!

Antwoord d: I1=−(Ib+Ic+Id)

  • Fout: Gebruikt Ib in plaats van I₂
  • Ib is niet gedefinieerd bij dit knooppunt

Praktijkvoorbeeld:

Stel I₂ = 3A, Ic = 2A, Id = 1A (alle weggaand van het knooppunt):

I1 = − ( 3 + 2 + 1 ) = − 6 A

Het negatieve teken betekent: I₁ stroomt naar het knooppunt (als we de conventie gebruiken dat uitgaande stromen positief zijn).

Controle: 6A komt binnen (van U₁), 3A + 2A + 1A = 6A gaat weg → balans klopt ✓

Fysische interpretatie

Spanningsbron versus stroombron:

Cruciaal verschil:

Type bron Wat wordt afgedwongen Wat wordt bepaald door netwerk
Spanningsbron Spanning U Stroom I (via KCL!)
Stroombron Stroom I Spanning U (via KVL!)

Dit verklaart waarom:

"De stroom door een spanningsbron wordt bepaald door de weerstanden en stromen in de rest van het netwerk. De spanningsbron 'levert' precies genoeg stroom om aan KCL te voldoen."

Energetische betekenis:

De stroom I₁ door de spanningsbron bepaalt het vermogen dat de bron levert:

P = U1 · I1

Als I₁ negatief is (stroom gaat "tegen" de spanningsbron in), levert de bron energie. Als I₁ positief is, absorbeert de bron energie (bijvoorbeeld bij opladen van een batterij).

Praktische toepassing:

Dit principe wordt gebruikt in:

  • Stroommeters: Meet de stroom door een spanningsbron om belasting te bepalen
  • Batterijbewaking: Bepaal of batterij ontlaadt (I < 0) of oplaadt (I > 0)
  • Netwerkvermogen: Bereken hoeveel vermogen een voeding moet leveren

Antwoord: b

I1=−(I2+Ic+Id)

De stroom door een spanningsbron wordt bepaald door Kirchhoff knooppuntwet toegepast op het aansluitpunt.
