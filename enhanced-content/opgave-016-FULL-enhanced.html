In dit netwerk is de stroom door de weerstand Rc gelijk aan:

a: Ic=(Ib−I3)−(I2+Id)

b: Ic=[(Ib−I3)−(I2+Id)]·[Rd/(Rc+Rd)]

c: Ic=(I3+I2)

d: Ic=−I1−I2−Id

Uitwerking
Stap 1: Analyseer de schakeling en gegeven informatie

Deze vraag test de Kirchhoff knooppuntwet op een complex knooppunt met meerdere stromen.

Element Beschrijving
Knooppunt B Waar I₁, I₂, Ic en Id samenkomen
Stroomen naar B I₁ (van stroombron)
Stromen van B weg I₂, Ic, Id
Gezochte stroom Ic door Rc

Theoretische basis: Kirchhoff knooppuntwet (KCL): ΣI=0 voor alle stromen naar/van een knooppunt (Holmes §1.8).

Stap 2: Identificeer alle stromen bij knooppunt B

Bij knooppunt B zijn de volgende stromen aanwezig (let op de richting!):

  • I1: stroomt naar B (positief)
  • I2: stroomt weg van B (negatief)
  • Ic: stroomt weg van B (negatief)
  • Id: stroomt weg van B (negatief)

Belangrijk: In dit netwerk is I₁ de stroom van de stroombron die naar knooppunt B stroomt. De stromen I₂, Ic en Id zijn de uitgaande stromen.

Stap 3: Pas Kirchhoff knooppuntwet toe

De Kirchhoff knooppuntwet zegt dat de som van alle stromen naar een knooppunt nul is:

I1 + I2 + Ic + Id = 0

Waarbij I₂, Ic en Id negatief zijn (weggaand).

Stap 4: Los op naar Ic

Isoleer Ic in de vergelijking:

Ic = − I1 − I2 − Id

Conclusie: Het juiste antwoord is d.

Stap 5: Verificatie en fysische interpretatie

Dimensionale controle:

[Ic] = [A] − [A] − [A] = [A]

Dit klopt ✓

Waarom andere antwoorden fout zijn:

Antwoord a: Ic=(Ib−I3)−(I2+Id)

  • Fout: Gebruikt Ib en I₃ die niet bij knooppunt B aansluiten
  • Dit zou mogelijk een uitdrukking zijn na substitutie van andere knooppuntvergelijkingen, maar niet de directe toepassing van KCL op knooppunt B

Antwoord b: Bevat stroomdelingsformule

  • Fout: Stroomdeling is niet relevant voor directe Kirchhoff toepassing

Antwoord c: Ic=(I3+I2)

  • Fout: Negeert I₁ en Id
  • Zou alleen kloppen als I₁ + Id = 0, wat niet algemeen geldt

Praktijkvoorbeeld:

Stel I₁ = 10A (naar B), I₂ = -3A (van B weg), Id = -2A (van B weg):

Ic = − 10 − ( − 3 ) − ( − 2 ) = − 10 + 3 + 2 = − 5 A

Controle: 10A komt binnen, 3A + 2A + 5A = 10A gaat weg → balans klopt ✓

Fysische interpretatie

Stroombalans in complexe netwerken:

De vergelijking Ic=−I1−I2−Id vertelt ons:

"De stroom door Rc is het restant van de binnenkomende stroom I₁ nadat I₂ en Id zijn afgetakt."

Of, herschreven als:

I1 = − I2 − Ic − Id

Dit zegt: "De totale binnenkomende stroom I₁ splitst zich in drie uitgaande stromen."

Belang van tekenconventie:

Bij complexe netwerken is het cruciaal om een consistente tekenconventie te hanteren:

  • Kies één richting als positief (bijv. "naar knooppunt")
  • Houd deze conventie aan voor ALLE stromen
  • Negatieve uitkomsten betekenen dat de werkelijke richting tegengesteld is

Antwoord: d

Ic=−I1−I2−Id

Directe toepassing van Kirchhoff knooppuntwet op knooppunt B.
