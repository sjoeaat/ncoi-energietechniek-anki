In dit netwerk is de stroom door de weerstand Rd gelijk aan:

a: Id=I2·[Rb·Rd/(Rb+Rd)]

b: Id=I2·[Rb/(Rb+Rd)]

c: Id=I2·[Rd/(Rb+Rd)]

d: Id=−I2−Ib

Uitwerking
Stap 1: Analyseer de gegeven informatie

Deze vraag test de directe toepassing van de Kirchhoff knooppuntwet zonder verdere vereenvoudiging.

Element Beschrijving
Knooppunt A Waar I₂, Ib en Id samenkomen
I₂ Stroom van buitenaf naar knooppunt A
Ib Stroom door Rb (van A weg)
Id Gezochte stroom door Rd (van A weg)

Theoretische basis: Kirchhoff knooppuntwet (KCL): ΣI=0 voor alle stromen naar/van een knooppunt (Holmes §1.8).

Stap 2: Pas Kirchhoff knooppuntwet toe op knooppunt A

Bij knooppunt A gelden de volgende stroomrichtingen:

  • I₂ stroomt naar knooppunt A → positief
  • Ib stroomt weg van knooppunt A → negatief
  • Id stroomt weg van knooppunt A → negatief

De Kirchhoff knooppuntwet geeft:

I2 + Ib + Id = 0

Waarbij Ib en Id negatief zijn omdat ze van het knooppunt weg stromen.

Stap 3: Los op naar Id

Isoleer Id in de vergelijking:

Id = − I2 − Ib

Conclusie: Het juiste antwoord is d.

Stap 4: Verificatie - waarom andere antwoorden fout zijn

Antwoord a, b en c: Deze gebruiken stroomdelingsformules, wat suggereert dat je eerst de parallelle weerstandscombinatie moet berekenen.

Maar de vraag vraagt directe relatie tussen Id, I₂ en Ib via Kirchhoff, NIET via stroomdeling.

Als je wél stroomdeling zou gebruiken (opgave 11 stijl), zou je eerst Ib moeten uitdrukken in termen van I₂ en Rb/Rd, en dan Id berekenen. Dat is NIET wat deze vraag vraagt.

Stap 5: Verificatie en fysische interpretatie

Dimensionale controle:

[Id] = [A] − [A] = [A]

Dit klopt ✓

Praktijkvoorbeeld:

Stel I₂ = 6A stroomt naar knooppunt A, en Ib = -4A (weggaand):

Id = − 6 − ( − 4 ) = − 6 + 4 = − 2 A

Controle: 6A komt binnen, 4A + 2A = 6A gaat weg → balans klopt ✓

Fysische interpretatie

Kirchhoff als stroombalans:

De vergelijking Id=−I2−Ib zegt fysisch:

"De stroom door Rd (weggaand van A) is gelijk aan de netto stroom die NIET door Rb gaat, nadat I₂ is binnengekomen."

Alternatieve interpretatie:

Herschrijf de vergelijking als:

I2 = − Ib − Id

Dit zegt: "De totale binnenkomende stroom I₂ splitst zich in twee uitgaande stromen: Ib en Id".

Antwoord: d

Id=−I2−Ib

Directe toepassing van Kirchhoff knooppuntwet op knooppunt A.
