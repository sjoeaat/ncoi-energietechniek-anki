H 9.7 - Opgave 10: Onbekende L en bekende R

Gegeven:

  • Serie RL-netwerk (R + L in serie)
  • R = 25 Ω (bekend)
  • Totale impedantie: |Z| = 100 Ω
  • Frequentie: f = 400 Hz

Vraag: Bereken L en φ

Uitwerking volgens 5-stappen methodologie
Stap 1: Analyseer de gegeven informatie

Netwerkconfiguratie:

  • Serie RL: Z = R + jωL
  • R = 25 Ω
  • |Z| = 100 Ω (modulus gegeven)
  • ω = 2πf = 2π × 400 = 800π rad/s ≈ 2513 rad/s

Te bepalen:

  • Inductantie L [H]
  • Fasehoek φ [rad of°]

Holmes referentie: H 9.7 - Complexe impedanties

Stap 2: Bepaal toe te passen theorie

Serie RL impedantie:

Z = R + jωL = R + j XL

Modulus (Pythagoras):

|Z| = R2 + XL2

XL oplossen:

XL = |Z| 2 − R2

Inductantie uit reactantie:

L = XL 2πf

Fasehoek:

φ = arctan XL R
Stap 3: Algebraïsche afleiding

Gegeven:

  • |Z|² = R² + XL²
  • 100² = 25² + XL²

Los op naar XL:

XL2 = 1002 − 252 = 10000 − 625 = 9375
XL = 9375 = 96,82 Ω

Bereken L:

L = XL 2πf = 96,82 2π · 400 = 96,82 2513 = 38,52 mH

Bereken φ:

φ = arctan 96,82 25 = arctan ( 3,873 ) = 1,318 rad = 75,52 °
Stap 4: Numerieke verificatie

Controle |Z|:

|Z| = 252 + 96,822 = 625 + 9374,11 = 9999,11 ≈ 100 Ω ✓

Controle XL uit L:

XL = 2π · 400 · 38,52 × 10−3 = 96,82 Ω ✓
Stap 5: Verificatie en fysische interpretatie

Dimensionale controle:

  • [XL] = √(Ω² - Ω²) = Ω ✓
  • [L] = [XL/(2πf)] = Ω / (s-1) = Ω·s = V·s/A = H ✓
  • [φ] = arctan(Ω/Ω) = dimensieloos ✓

Fysische betekenis:

Impedantie-driehoek:

Voor serie RL vormen R, XL, en |Z| een rechthoekige driehoek:

  • Horizontaal: R = 25 Ω (resistief)
  • Verticaal: XL = 96,82 Ω (inductief)
  • Hypotenusa: |Z| = 100 Ω
  • Hoek: φ = 75,52° (stroom loopt achter)

Verhouding XL/R ≈ 4:

Het netwerk is sterk inductief:

  • XL ≈ 4R betekent dat reactieve component domineert
  • Arbeidsfactor: cos φ = R/|Z| = 25/100 = 0,25 (zeer laag!)
  • Slechts 25% van de schijnbaar vermogen is actief vermogen
  • 75% is reactief vermogen (oscillerend, niet gedissipeerd)

Fasehoek φ = 75,52°:

Een grote fasehoek betekent:

  • Stroom loopt fors achter op spanning (bijna 90°)
  • Het netwerk gedraagt zich bijna als een pure spoel
  • Bij pure L zou φ = 90°, hier is het 75,5° door R

Inductantie L = 38,52 mH:

Dit is een behoorlijk grote inductie voor 400 Hz:

  • Bij 50 Hz zou XL = 2π·50·0,03852 ≈ 12 Ω (veel kleiner)
  • Bij 10 kHz zou XL ≈ 2420 Ω (veel groter)
  • XL ∝ f: Inductieve reactantie groeit lineair met frequentie

Praktische context:

Serie RL-netwerken met hoge XL/R ratio komen voor in:

  • Transformators: Primaire wikkeling bij lege secundair
  • Relais- en motorspoelen: Hoge inductie, lage R
  • RF-chokes: Hoge impedantie bij bepaalde frequenties
  • Ballasts: TL-armaturen (oude stijl)

Vermogenskarakter:

Bij lage arbeidsfactor (0,25):

  • Hoge stroom voor weinig actief vermogen
  • Grote verliezen in distributie
  • Power factor correctie noodzakelijk (parallel C toevoegen)
  • Compensatiecondensator zou QC = -QL moeten leveren

Ontwerp-implicaties:

Als |Z| en R bekend zijn, kan L eenvoudig berekend worden via Pythagoras. Dit is handig bij:

  • Spoelkarakterisering (meten R en |Z|, bereken L)
  • Kwaliteitsfactor bepaling: Q = XL/R ≈ 3,87 (matige Q)
  • Filter ontwerp (bepaal benodigde L voor gewenste |Z|)

Antwoord:

L = 38,52 mH

φ = 1,318 rad (75,52°)

Verificatie: XL = √(100² - 25²) = 96,82 Ω

Fysisch: Het serie RL-netwerk is sterk inductief (XL ≈ 4R), met een arbeidsfactor van slechts 0,25. De grote fasehoek (75,5°) betekent dat de stroom bijna 90° achterloopt op de spanning, vergelijkbaar met een bijna-pure spoel. De kwaliteitsfactor Q = XL/R ≈ 3,87 duidt op matige verliezen in de spoel.
