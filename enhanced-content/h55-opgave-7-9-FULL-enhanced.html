H 5.5 - Opgave 7-9: Vectoriële optelling van wisselspanningen

Tel meerdere AC-signalen met verschillende fases vector ieel op in het complexe vlak.

Gegeven: Meerdere sinusoïdale signalen met verschillende amplitudes en fasehoeken

Gevraagd: Resulterende signaal na vectoriële optelling

Methode: Euler formule en complexe getallen

Uitwerking
Stap 1: Theoretische basis - Euler formule

Deze opgaven testen het begrip van fasorrekenen en het superpositieprincipe voor AC-signalen.

Euler formule:

eiφ = cos(φ) + i · sin(φ)

Conversie tijddomein ↔ complex:

Tijddomein Complex (fasor) Hoek
A·cos(ωt + φ) A·eiφ φ radialen
A·sin(ωt + φ) A·ei(φ - π/2) φ - 90° radialen

Waarom werkt dit? sin(x) = cos(x - π/2), dus een sinus is een cosinus met -90° faseverschuiving.

Stap 2: BELANGRIJKE REGEL - Gelijknamig maken

Regel: Alle signalen moeten "gelijknamig" zijn voordat je ze op kunt tellen!

"Je kunt een cos() niet optellen bij een sin() - net zoals je appels niet bij peren kunt optellen. Eerst alles naar hetzelfde type converteren!"

Conversiestappen:

Als de uitkomst in cos() moet zijn:

  • Cosinus: geen aanpassing
  • Sinus: trek π/2 af van de fasehoek φ

Als de uitkomst in sin() moet zijn:

  • Sinus: geen aanpassing
  • Cosinus: tel π/2 op bij de fasehoek φ

Voorbeeld:

5·sin(ωt + 30°) omzetten naar cosinus:

5 · sin(ωt+30°) = 5 · cos(ωt+30°−90°)
= 5 · cos(ωt−60°)
Stap 3: Procedure voor vectoriële optelling

Complete procedure (5 stappen):

Stap 3a: Kies norm (cos of sin)

Besluit: alle signalen worden cosinussen (cos is de norm)

Stap 3b: Converteer alle sinussen naar cosinussen

Voor elke sin(ωt + φ): vervang door cos(ωt + φ - π/2)

Stap 3c: Converteer naar complex met Euler

Voor elk signaal A·cos(ωt + φ):

U→n = A · eiφ = A · [ cos(φ) + i · sin(φ) ]

Stap 3d: Tel alle complexe getallen op

U→tot = ∑ n=1 N U→n

Stap 3e: Converteer terug naar tijddomein

Bereken amplitude en fase uit het resulterende complexe getal:

A = (Re)2 + (Im)2
φ = arctan Im Re

Resultaat: A·cos(ωt + φ)

Stap 4: Praktijkvoorbeeld (Excel implementatie)

Voorbeeld signalen:

  • U₁ = 10·cos(ωt + 0°)
  • U₂ = 5·sin(ωt + 45°)
  • U₃ = 8·cos(ωt + 90°)

Stap 1: Gelijknamig maken (alles naar cos):

  • U₁ = 10·cos(ωt + 0°) → blijft hetzelfde
  • U₂ = 5·sin(ωt + 45°) = 5·cos(ωt + 45° - 90°) = 5·cos(ωt - 45°)
  • U₃ = 8·cos(ωt + 90°) → blijft hetzelfde

Stap 2: Complex projectie met Euler:

Signaal φ (rad) eiφ Re (cos φ) Im (sin φ) Fasor
U₁ 0 ei·0 = 1 10 0 10 + 0i
U₂ -π/4 e-iπ/4 3.54 -3.54 3.54 - 3.54i
U₃ π/2 eiπ/2 = i 0 8 0 + 8i

Stap 3: Optelling:

U→tot = (10+3.54+0) + i (0−3.54+8)
= 13.54 + 4.46i

Stap 4: Terug naar tijddomein:

A = 13.542 + 4.462 = 14.26
φ = arctan 4.46 13.54 = 18.2°

Resultaat: Utot = 14.26·cos(ωt + 18.2°)

Stap 5: Verificatie en fysische interpretatie

Controle: Is Atot ≤ ΣAi? → 14.26 ≤ (10+5+8) = 23 ✓

(Bij destructieve interferentie kan de totale amplitude kleiner zijn dan de som)

Fysische interpretatie

Waarom complexe getallen gebruiken?

"Complexe getallen zijn NIET abstract - ze zijn een geometrische representatie van roterende vectoren in het 2D-vlak. De reële as is 'cos', de imaginaire as is 'sin'."

Grafische betekenis:

  • Reële as (Re): Component in fase met referentie (0°)
  • Imaginaire as (Im): Component 90° voor op referentie
  • Amplitude |U|: Lengte van de vector
  • Fase φ: Hoek t.o.v. reële as

Praktische toepassingen:

  • Driefasesystemen: Fasoren van 120° verschoven optellen
  • Filter ontwerp: Frequentierespons als complexe functie
  • Impedantie berekeningen: Z = R + jX
  • AC vermogensanalyse: S = P + jQ (complex vermogen)

Waarom Excel gebruiken?

Excel heeft de functie COMPLEX.EXP() die automatisch eiφ berekent:

  • COMPLEX.EXP(0) = 1 (cos 0°)
  • COMPLEX.EXP(π/2) = i (cos 90° = 0, sin 90° = 1)
  • COMPLEX.EXP(-π/2) = -i (cos -90° = 0, sin -90° = -1)

Dit maakt de berekeningen veel sneller en minder foutgevoelig!

Methode samenvatting:

  1. Kies norm: cos of sin
  2. Maak alle signalen "gelijknamig"
  3. Converteer naar complex: A·eiφ
  4. Tel complexe getallen op: Σ(Re + i·Im)
  5. Converteer terug: A = √(Re²+Im²), φ = arctan(Im/Re)

Belangrijkste valkuil: Vergeet NIET om sinussen te converteren!

Euler formule is de sleutel tot eenvoudige AC-analyse!
