In dit netwerk is de stroom door de weerstand Rb gelijk aan:

a: Ib=−I2·Rd/RbI_{b} = {- I}_{2} \bullet R_{d}/R_{b}

c: Ib=−I2·Rd/(Rb+Rd)I_{b} = {- I}_{2} \bullet R_{d}/{(R}_{b} + R_{d})

Uitwerking
Stap 1: Analyseer de schakeling en gegeven informatie

Deze vraag test begrip van stroomverdeling in parallelgeschakelde weerstanden en de wet van Kirchhoff voor knooppunten (KCL).

Element Symbool Rol
Stroom van buitenaf I2 Stroomt naar knooppunt A
Weerstand b Rb Parallel met Rd
Weerstand d Rd Parallel met Rb
Gezochte stroom Ib Stroom door Rb

Theoretische basis:

  • Kirchhoff Knooppuntwet (KCL): De som van alle stromen naar een knooppunt is nul: ΣI=0 (Holmes §1.8)
  • Wet van Ohm: Voor weerstanden geldt U=I·R
  • Stroomdeling: Bij parallelschakeling verdelen stromen zich omgekeerd evenredig met de weerstandswaarden (lagere weerstand → hogere stroom)
Stap 2: Pas Kirchhoff Knooppuntwet toe op knooppunt A

Bij knooppunt A komen/gaan drie stromen:

  • I2: stroomt naar knooppunt A (positief)
  • Ib: stroomt van knooppunt A weg (negatief)
  • Id: stroomt van knooppunt A weg (negatief)

De Kirchhoff knooppuntwet geeft:

I2 + Ib + Id = 0

Fysische betekenis: Alle stroom die binnenkomt moet ook weer weggaan. De stroom I₂ splitst zich over de twee paralleltakken Rb en Rd.

Stap 3: Gebruik wet van Ohm voor gelijke spanningen

Omdat Rb en Rd parallel staan, geldt dat de spanning over beide weerstanden gelijk is:

Ub = Ud

Volgens de wet van Ohm kan dit geschreven worden als:

Ib · Rb = Id · Rd

Los op naar Id:

Id = Ib · Rb Rd
Stap 4: Combineer vergelijkingen en los op naar Ib

Substitueer de uitdrukking voor Id in de Kirchhoff vergelijking:

I2 + Ib + Ib · Rb Rd = 0

↓ (factoriseer Ib)

I2 + Ib · Rd Rd + Ib · Rb Rd = 0

↓ (combineer breuken)

I2 + Ib · Rd + Rb Rd = 0

↓ (isoleer Ib)

Ib · Rd + Rb Rd = − I2

↓ (vermenigvuldig met omgekeerde breuk)

Ib = − I2 · Rd Rd + Rb

Of geschreven als:

Ib = − I2 · Rd / ( Rb + Rd )

Conclusie: Het juiste antwoord is c.

Stap 5: Verificatie en fysische interpretatie

Dimensionale controle:

[ Ib ] = [ A ] · [Ω] [Ω] = [ A ]

Dit klopt ✓

Analyse van de andere antwoorden:

  • Antwoord a: Ib=−I2·Rd/Rb — Fout: dimensie klopt niet (geeft [A]·[Ω]/[Ω] maar zonder juiste verhouding)
  • Antwoord b: Fout: Rb in teller i.p.v. Rd
  • Antwoord d: Fout: gebruikt Ic en Rc die niet relevant zijn voor deze tak
Fysische interpretatie

Stroomdeling in parallelschakeling:

De formule Ib=−I2·Rd/(Rb+Rd) is de stroomdelingsformule.

Belangrijke observaties:

  • Het minteken geeft aan dat Ib de tegengestelde richting heeft t.o.v. I₂ (I₂ komt binnen, Ib gaat weg)
  • Omgekeerde evenredigheid: De stroom door Rb is evenredig met de andere weerstand (Rd), niet met zijn eigen weerstand
  • Als Rd = 0Ω (kortsluiting): Ib = 0A (alle stroom gaat door de kortsluiting)
  • Als Rd → ∞ (open keten): Ib → −I₂ (alle stroom gaat door Rb)

Praktijkvoorbeeld:

Stel I₂ = 6A, Rb = 2Ω, Rd = 4Ω:

Ib = − 6 · 4 2+4 = − 6 · 4 6 = − 4 A

En:

Id = − I2 − Ib = − 6 − ( − 4 ) = − 2 A

Controle: De lagere weerstand (Rb = 2Ω) krijgt de hogere stroom (4A vs 2A). Ratio: 4A/2A = 2, wat overeenkomt met omgekeerde weerstandsverhouding 4Ω/2Ω = 2 ✓

Antwoord: c

Ib=−I2·Rd/(Rb+Rd)

Stroomdelingsformule voor parallelweerstanden met Kirchhoff knooppuntwet.
