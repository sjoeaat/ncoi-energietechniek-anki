H 3.11 - Opgave 2d

Vereenvoudig dit netwerk tot Thévenin en Norton equivalenten.

Gegeven: 5V spanningsbron, twee 10Ω weerstanden (parallel), 2A stroombron

Uitwerking
Stap 1: Analyseer de schakeling

Strategie: Twee methoden mogelijk - (1) intuïtief via bron-conversie, (2) analytisch via vergelijkingen

Stap 2: Bereken Rth
Rth = 10Ω ∥ 10Ω = 5Ω
Stap 3: Bereken Uth (methode 1 - intuïtief)

Stap 3a: Converteer 2A stroombron parallel met 10Ω naar spanningsbron:

U = 2 × 10 = 20 V

Stap 3b: Spanningsverschil over de twee 10Ω weerstanden:

Δ U = 20V - 5V = 15V over 2×10Ω → elk krijgt 7.5V

Stap 3c: Thévenin spanning:

Uth = 20 − 7.5 = 12.5 V

(of van onderen: 5V + 7.5V = 12.5V)

Stap 4: Bereken Ik
Ik = 12.5 5 = 2.5 A
Stap 5: Verificatie (alternatieve methode)

Maas vergelijking:

5V - I₁·10Ω - I₂·10Ω = 0

Knooppunt: I₁ - I₂ + 2A = 0 → I₁ = I₂ - 2

Substitutie: 5 - 10(I₂-2) - 10I₂ = 0 → 25 = 20I₂ → I₂ = 1.25A

Uth = 5V - I₁·10Ω = 5V + 7.5V = 12.5V ✓

Thévenin: 12.5V in serie met 5Ω

Norton: 2.5A parallel met 5Ω
