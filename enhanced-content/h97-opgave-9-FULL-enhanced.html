H 9.7 - Opgave 9: Spoel in serie met R || (R + C)

Gegeven:

  • Spoel: L = 10 mH in serie met bron
  • Parallel blok: R1 = 100 Ω || [R2 = 50 Ω + C = 5 μF in serie]
  • Frequentie: f = 1 kHz

Vraag: Bereken totale impedantie |Z| en fasehoek φ

Uitwerking volgens 5-stappen methodologie
Stap 1: Analyseer de gegeven informatie

Netwerktopologie:

  • Serie-element: L = 10 mH = 0,010 H
  • Parallel rechts: R1 || (R2 + C)
  • R1 = 100 Ω
  • R2 = 50 Ω
  • C = 5 μF = 5 × 10-6 F
  • f = 1000 Hz → ω = 2π × 1000 = 2000π rad/s

Te berekenen:

  • Totale impedantie Ztotaal
  • Modulus |Z|
  • Fasehoek φ

Holmes referentie: H 9.7 blz. 120 - Complexe RLC-netwerken

Stap 2: Bepaal toe te passen theorie

Complex impedantieberekening stappenplan:

  1. Bereken ZL = jωL
  2. Bereken ZC = 1/(jωC) = -j/(ωC)
  3. Bereken Zrechts = R1 || (R2 + ZC)
  4. Bereken Ztotaal = ZL + Zrechts
  5. Modulus en fasehoek bepalen

Parallel impedantie formule:

Zparallel = Z1 · Z2 Z1 + Z2

Complex delen (rationaliseren):

Bij deling door complex getal a + bj: vermenigvuldig teller en noemer met complex toegevoegde (a - bj)

Stap 5: Verificatie en fysische interpretatie

Dimensionale controle:

  • [ZL] = [ωL] = s-1 · H = s-1 · (V·s/A) = V/A = Ω ✓
  • [ZC] = [1/(ωC)] = (s-1 · F)-1 = (s-1 · A·s/V)-1 = V/A = Ω ✓
  • [|Z|] = √(Ω² + Ω²) = Ω ✓
  • [φ] = dimensieloos (radialen) ✓

Sanity checks:

  • XL ≈ 62,8 Ω ✓
  • XC ≈ 31,8 Ω ✓
  • Zrechts heeft Re > 0, Im < 0 ✓ (netto capacitief in parallel blok)
  • Ztotaal heeft Im > 0 ✓ (netto inductief door serie-L)
  • |Z| > |Re| ✓
  • φ > 0 ✓ (inductief gedrag dominant)

Fysische betekenis:

Netwerk karakter - Netto inductief:

De totale impedantie Z = 36,20 + 49,29j Ω heeft:

  • Re(Z) = 36,20 Ω: Dissipatie (beide R-en)
  • Im(Z) = +49,29 Ω: Netto inductief (XL > XC,eff)
  • φ = 53,7° > 0: Stroom loopt achter op spanning

Parallel blok effect:

Het parallelle blok Zrechts = 36,20 - 13,54j Ω heeft:

  • Capacitief karakter (Im < 0)
  • Dit komt door de C in de (R+C)-tak
  • De 100Ω parallel-R verhoogt het resistieve deel

Serie-L dominant:

Ondanks het capacitieve parallel blok (Im = -13,54 Ω), is de serie-spoel (XL = +62,83 Ω) dominant, waardoor het totale netwerk inductief gedraagt:

Im ( Ztotaal ) = 62,83 − 13,54 = 49,29 Ω > 0 (inductief)

Vermogenskarakter:

  • Arbeidsfactor: cos φ = Re(Z)/|Z| = 36,20/61,2 ≈ 0,592 (achterblijvend)
  • Reactief vermogen: Positief (netwerk absorbeert reactief vermogen als inductieve last)
  • Vermogensdriehoek: P = 59,2%, Q = 80,5% (inductief), S = 100%

Frequentieafhankelijkheid:

Bij f = 1 kHz:

  • XL = 2πfL ∝ f (lineair met frequentie)
  • XC = 1/(2πfC) ∝ 1/f (omgekeerd evenredig)
  • Bij hogere f: XL ↑, XC ↓ → meer inductief
  • Bij lagere f: XL ↓, XC ↑ → meer capacitief

Praktische context:

Dit type netwerk (serie-L met parallel RC-blok) komt voor in:

  • Impedantie-matching: Aanpassing tussen bron en last
  • Audio-crossovers: Weef/mid-range scheidingsfilters
  • RF-circuits: Aanpassingsnetwerken voor antennes
